# world_time_viewer
This is a small project designed for my practicing Behavior-Driven Development (BDD), pytest and PySide6.

## How to run
This app made with Python 3.10.12.
Execute the following command:
'''
python src/main.py
'''

  |    |- world_time_viewer.feature
  |- README.md
